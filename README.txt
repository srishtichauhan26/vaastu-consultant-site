Shubh Vaastu — Multi‑page demo website

How to use:
1) Download the ZIP.
2) Extract it.
3) Open index.html in any modern browser.
4) Edit text, prices, contact info as needed.
5) To host, upload the folder to GitHub Pages, Netlify, or any static host.

Pages:
- index.html (Home)
- about.html (About)
- services.html (Services)
- portfolio.html (Case Studies)
- blog.html (Guides)
- contact.html (Contact)

Tech: HTML, CSS, JavaScript (no frameworks). Responsive, dark/light mode, sticky header, carousel, scroll animations, and client-side form handler.
